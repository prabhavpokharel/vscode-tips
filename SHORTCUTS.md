# VS Code Keyboard Shortcuts Cheat Sheet

> Print this out or keep it open in a second tab. Refer to it until the shortcuts become muscle memory.

---

## Navigation

| Shortcut (Win/Linux) | Shortcut (Mac) | Action |
|----------------------|----------------|--------|
| `Ctrl+P` | `Cmd+P` | Open file by name / Command Palette |
| `Ctrl+Shift+P` or `F1` | `Cmd+Shift+P` or `F1` | Run a command |
| `Ctrl+G` | `Ctrl+G` | Go to line number |
| `Ctrl+Shift+.` | `Cmd+Shift+.` | List symbols in current file |
| `Ctrl+0` | `Cmd+0` | Focus sidebar |
| `Ctrl+1` | `Cmd+1` | Focus editor |
| `Ctrl+B` | `Cmd+B` | Toggle sidebar |
| `Ctrl+Tab` | `Ctrl+Tab` | Switch between open tabs |
| `Alt+1/2/3` | `Cmd+1/2/3` | Jump directly to tab 1, 2, or 3 |
| `Ctrl+Shift+E` | `Cmd+Shift+E` | Focus file explorer |
| `Ctrl+Shift+X` | `Cmd+Shift+X` | Open extensions panel |

| `Ctrl+Z` | `Cmd+Z` | Undo |
| `Ctrl+Shift+Z` | `Cmd+Shift+Z` | Redo |

| `Shift+Tab` | `Shift+Tab` | Outdent |

## Tips for Learning These

- **Don't memorise everything at once.** Pick 2–3 shortcuts per week and use them until they feel automatic.
- **Print this sheet.** Keep it on your desk for the first few weeks.
- **Customise anything.** Go to `File → Preferences → Keyboard Shortcuts` to change any shortcut to whatever feels natural.
- **Discover shortcuts in menus.** Every VS Code menu item shows its shortcut right next to it — a great way to find new ones while you work.
